# 📝 To-Do List App

A simple and responsive Todo List application built using HTML, CSS, and JavaScript.

## 🚀 Features

- Add tasks
- Delete tasks
- Mark tasks as completed
- Data stored in local storage

## 🛠️ Tech Stack

- HTML
- CSS
- JavaScript
